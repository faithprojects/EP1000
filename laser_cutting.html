Laser Cut Musical Box

Fusion360 Parameters

Firstly I created parameters to set the measurements that I can easily use for later by calling the name of the parameter which makes changing the measurements easier to do.

  • Length, Width, Height and Thickness are the measurements for the size of the box and how thick the plywood is.
  • LSTab, WSTab and HSTab are the number of tabs that I want
  • LTabForm, WTabForm and HTabForm are the formulas to calculate the no. of tabs equally separated
  • LPattForm, WPattForm and HPattForm are the formulas for the distance of the rectangle pattern for the finger joints
  • LidInnerHole is the size of the inner hole at the front body for the moveable lid.
  • LHandle and WHandle is the measurements for thee size of the hole for the handle to fit.

Base Body

  1. Starting with the base of the box I first made a box using the user parameters i put the length, width and heigh of the box.
  2. Create a new sketch and using a 2 point rectangle, draw the shape of the tab and using LTabForm and Thickness parameters. Use the sketch dimension to space out the tab.
  3. Using Extrude tool, cut the rectangle shape you made to make the tab.
  4. Then using the rectangular pattern, using the features type click on to the rectangle shape that you cut in Step 3 and select the direction that is highlighted in blue then use LPattForm which is the distance and quantity is LSTab parameters.
  5. Following the same steps from 1 to 4, do it on the width side of the base.

Front body

  1. Now we will start sketching the front so place a new sketch infront of the tab and start creating your front body using your Length and Height parameters. I used the center circle and offset tool to make the circles and hole for the moveable lid later.
  2. Then extrude the front body sketch using the same thickness parameters. Make syure it is a new body.
  3. As you have noticed the front body does not have the tabs that were created on the base. In order for the front body to have the tabs, use the combine tool with the target body being the front and the tool body the base and remember to tick the keep tools option. Now your front body will have tabs.
  4. Follow the same steps as the front body to create the tabs for the height.

Back Body

  1. To create back body similar to the front, using extrude tool start from the other end of the length side where the back will be and extrude the distance using Thickness parameter and make sure it is a new body.
  2. Using combine tool, target body is the base while tool body is the back to make tab at the base.

Left Body

  1. Start with a new sketch on the left and start creating your left body using width and height parameters and then extrude the sketch using the same thickness parameters. Making sure it is a new body.
  2. Using the combine tool, target body the left body that you have just extruded while tool body is the rest of the other bodies to make the tabs on the left body.

Right Body

  1. Using the extrude tool, to create the right body similar to the left body, start from the other end of the right side and extrude the distance using Thickness parameter and make sure it is a new body.
  2. Using combine tool, target body the base and tool body the newly created right body to make a tab for the right side of the base
  3. Using combine tool, target body the front and tool body the newlyy created right body to make a tab for the right side of the front
  4. Do the same for the back body.

Hole for Handle

  1. To make the hole for the handle, I started a new sketch on the right body and using a 2 point rectangle, I made a rectangle using LHandle and WHandle parameters.
  2. Then extrude the rectangle to make a hole.
